# vk-blocks

This is a plugin that extends Gutenberg's blocks.

## How to Build

```
npm install
npm run build
npm run watch
```

## Translate Only

```
npm run build
```

## CSS name rule

```
[ prefix ]_[ block name ]_[section name]
[ prefix ]_[ block name ]_[section name]_[sub section name]
[ prefix ]_[ block name ]-[Attribute name]-[Attribute property]
```
